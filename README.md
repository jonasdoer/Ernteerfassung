# Ernteerfassung Website

Diese statische Website kann direkt über GitHub Pages veröffentlicht werden.

## Hochladen
1. Neues GitHub-Repository erstellen.
2. `index.html`, `style.css` und `script.js` hochladen.
3. Unter **Settings → Pages** als Source den Branch `main` und `/root` auswählen.
4. Speichern. Danach ist die Website über die angezeigte GitHub-Pages-Adresse erreichbar.

## Hinweis
Die Eingaben werden im Browser per LocalStorage gespeichert. Für mehrere Geräte oder Benutzer ist später eine Datenbank nötig.
